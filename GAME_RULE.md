# 게임 규칙: 하늘에서 과일이 비처럼 내려와 🍎

> 3개의 라인에서 떨어지는 과일을 바구니로 받아내는 게임입니다.

## 🎮 게임 제목

**Sky Fruit Catcher (하늘 과일 담기)**

---

## 📝 게임 설명

하늘에서 다양한 과일이 떨어집니다!
플레이어는 **왼쪽 / 가운데 / 오른쪽** 3개 구역으로 이동하며 과일을 받아야 합니다.
중간중간 떨어지는 **폭탄**을 피하세요!

---

## 🎯 게임 목표

- 제한 시간 동안 최대한 많은 과일을 받아 점수를 획득하세요.
- 폭탄을 피하고 생존하세요.
- **배수 아이템(황금 사과)**를 먹으면 고득점을 노릴 수 있습니다!

---

## 🕹️ 게임 규칙

### 기본 플레이

1. **화면 구성**: 화면은 3개의 세로 라인(LEFT, CENTER, RIGHT)으로 나뉩니다.
2. **조작**: 플레이어의 포즈에 따라 바구니가 이동합니다.
3. **획득**: 바구니와 과일이 닿으면 점수를 획득합니다.
4. **회피**: 폭탄에 닿으면 게임이 즉시 종료됩니다(또는 라이프 차감).

### 상세 규칙

- **이동 방식**: 사람의 몸 동작을 인식하여 이동
    - 왼쪽 기울이기/손들기 → **Left** 라인으로 이동
    - 정면 서기 → **Center** 라인으로 이동
    - 오른쪽 기울이기/손들기 → **Right** 라인으로 이동
- **콤보 시스템**: 과일을 놓치지 않고 연속으로 받으면 추가 점수 (+10, +20...)

## 🎨 필요한 포즈 목록 (Teachable Machine 학습용)

| 포즈 이름 (Label) | 설명 | 게임 동작 |
|:---:|---|---|
| **Left** | 몸을 왼쪽으로 기울이거나 왼손을 듦 | 바구니를 왼쪽 칸으로 이동 |
| **Center** | 차렷 자세 또는 정면 응시 | 바구니를 가운데 칸으로 이동 |
| **Right** | 몸을 오른쪽으로 기울이거나 오른손을 듦 | 바구니를 오른쪽 칸으로 이동 |

> **팁**: 'Center' 포즈는 아무것도 안 하고 서 있는 편안한 상태로 학습시키세요.

## ⏱️ 레벨 디자인

시간이 지날수록 게임이 어려워집니다.

| 단계 | 시간 | 변화 내용 |
|---|---|---|
| **Lv.1** | 0~20초 | 천천히 떨어짐, 폭탄 없음 |
| **Lv.2** | 20~40초 | 속도 **1.5배** 증가, 폭탄 등장 시작 |
| **Lv.3** | 40~60초 | 속도 **2.0배** 증가, 폭탄 빈도 증가 |
| **Fever** | 50초~ | 과일이 쏟아짐 (점수 2배) |
